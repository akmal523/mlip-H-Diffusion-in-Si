FILE DESCRIPTIONS
=================

--- diffusion_summary.txt ---
Plain-text summary of the full analysis.
Contains:
  * Diffusion coefficient D [cm²/s] and linear-fit R² at each temperature
  * Activation energy Ea [eV] from Arrhenius fit
  * Pre-exponential factor D0 [cm²/s]
  * Unit conversion notes
  * List of all output files

--- diffusion_summary.csv ---
Columns:
  T_K               — simulation temperature [K]
  D_A2_per_ps       — diffusion coefficient [Å²/ps]    (raw LAMMPS units)
  D_cm2_per_s       — diffusion coefficient [cm²/s]    (SI-adjacent, standard)
  R2                — R² of the linear fit to MSD vs time (closer to 1 = better)
  slope_A2_ps       — slope of MSD vs time fit [Å²/ps]
  slope_stderr      — standard error of the slope (scipy linregress)
  fit_t_start_ps    — start of linear-fit window [ps]
  fit_t_end_ps      — end of linear-fit window   [ps]
  timestep_ps       — timestep used in that run  [ps]
  n_points          — number of MSD data points read

  Unit note:
    D [cm²/s] = D [Å²/ps] × 1e-4
    (1 Å² = 1e-20 m², 1 ps = 1e-12 s → 1 Å²/ps = 1e-8 m²/s = 1e-4 cm²/s)

--- msd_{T}K.png ---
Plot of mean-squared displacement of the H atom vs simulation time [ps].

  Blue line  — MSD recorded by LAMMPS every 1000 steps
  Red dashed — linear fit:  MSD(t) = 6D·t + offset
  Red shading — region used for the fit (last 50% of the trajectory)

  The fit region avoids the ballistic regime at early times (t ≲ 1 ps)
  and any saturation at very late times if H reaches a defect.

  From the Einstein relation in 3D:
    MSD(t) = 6 D t
    D = slope / 6   [Å²/ps]

  A well-behaved diffusion run shows:
    * Short ballistic regime at t → 0 (steep, non-linear)
    * Long linear regime (diffusive plateau)
    * R² of linear fit > 0.90

  At low temperatures (700–800 K) the H atom may hop only a few times;
  MSD may look step-like rather than smoothly linear. This is normal for
  a single atom and requires very long runs for good statistics.

--- arrhenius.png ---
Arrhenius plot: log₁₀(D) on the y-axis, 1000/T [K⁻¹] on the x-axis.

  Blue dots  — simulated D values at each temperature
  Red line   — linear fit

  Physics:
    D(T) = D0 · exp(-Ea / kB T)
    log10(D) = log10(D0)  −  [Ea / (kB · ln10)] · (1/T)

  Fitting log10(D) vs (1000/T):
    slope     = -Ea · 1000 / (kB · ln10)
    intercept = log10(D0)

    → Ea [eV] = -slope × kB × ln10 × 1e-3
    → D0      = 10^intercept  [cm²/s]

  where kB = 8.617333e-5 eV/K.

  A straight line in this plot confirms Arrhenius behaviour (thermally
  activated hopping). Deviation from linearity would suggest a change
  in diffusion mechanism across the temperature range.


RAW SIMULATION OUTPUT (in diffusion_runs/)
==========================================

For each temperature, the directory  diffusion_runs/Si64H1_box/T{T}K/
contains:

  in.diffusion       — LAMMPS input script (auto-generated by run_diffusion.py)
  si_with_h.lmp      — copy of the structure file
  log.lammps         — LAMMPS console output (thermo data)
  msd_{T}K.dat       — MSD time series
                       columns: step  msd_total  msd_x  msd_y  msd_z  [Å²]
                       time [ps] = step × timestep
  dump.atom          — full trajectory (every 1000 steps)
                       columns per atom: id type x y z xu yu zu
                       x y z   = wrapped coordinates
                       xu yu zu = unwrapped (PBC-corrected, for MSD verification)
  final_state_{T}K.lmp — atomic positions at end of run


HOW TO REPRODUCE
================

1.  Set up environment:
      conda env create -f grace.yml
      conda activate lammps_grace_env

2.  Edit run_diffusion.py: set LAMMPS_EXE and GRACE_PATH.

3.  Run simulations (production mode):
      # In run_diffusion.py, set:  MODE = "production"
      python run_diffusion.py

4.  Wait for all jobs to finish:
      for d in diffusion_runs/Si64H1_box/T*/; do
          echo "=== $d ==="; tail -3 "$d/log.lammps"
      done

5.  Run analysis:
      python analyze_msd.py
      # Results appear in  results/


PHYSICAL INTERPRETATION
========================

Expected activation energies for H in/on Si:
  Surface diffusion     ~0.1 – 0.5 eV   (fast, low barrier)
  Subsurface/bulk hop   ~1.0 – 2.0 eV   (slower, thermally activated)

If Ea is much smaller than 0.5 eV, H is likely mobile even at 700 K and
the low-T runs may not sample enough hops for a reliable D.

If Ea > 1.5 eV, H is essentially immobile at 700–800 K; those runs
will show a flat MSD. Increase the temperature range or use longer runs.

For a single H atom, statistical noise in the MSD is inherent — consider
running multiple independent trajectories (different velocity seeds) and
averaging the MSD curves for more robust D values.
